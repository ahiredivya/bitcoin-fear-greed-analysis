# Trader Performance & Bitcoin Market Sentiment Analysis

# Overview : 
This project explores the relationship between trader performance and Bitcoin market sentiment. By analyzing historical trader data alongside the Fear & Greed Index, we aim to uncover patterns that can drive smarter trading strategies.

# Objectives
- Analyze how market sentiment affects trader behavior and profitability.
- Identify patterns of profitable vs. unprofitable trades under different market conditions.
- Visualize trends and correlations between sentiment and trade performance.

# Steps : 
# 1. Load the datasets: Import trader and sentiment data using pandas.
# 2.Data Cleaning:
- Handle missing values in closedPnL and other key columns.
- Ensure date columns are properly formatted.
# 3.Feature Engineering:
- Convert timestamps to dates.
- Merge trader data with sentiment data on date.
# 4.Exploratory Data Analysis (EDA):
- Analyze distribution of trade PnL.
- Visualize Fear & Greed Index over time.
- Explore correlations between sentiment and trade performance.
# 5.Insights & Patterns:
- Identify when traders are more likely to profit/loss.
- Detect trends and potential strategies influenced by market sentiment.

# Tools & Libraries
- Python 
- Pandas, NumPy for data manipulation
- Matplotlib, Seaborn for visualization
